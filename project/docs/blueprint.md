# **App Name**: SliceView

## Core Features:

- Shape Input: Canvas that supports both freehand drawing and selection from a set of pre-defined shapes or words.
- 3D Printer Connection Simulation: UI element, such as a button, to simulate connecting to a 3D printer, providing on/off status.
- Print Preview: Display of a 3D preview of the selected or drawn shape with X, Y, and Z axes.
- Layer Simulation: Visualization of the printing process layer by layer using dummy or mock data.
- Verbose Feedback: Dynamically updated textual feedback on the printing process steps (e.g., 'Heating nozzle,' 'Extruding filament,' etc.).
- Project Information Button: Simple information button that reveals a modal or sidebar with project details and team member names, populated with dummy content.

## Style Guidelines:

- Primary color: Calm, desaturated blue (#64B5F6) for a modern and trustworthy feel.
- Background color: Very light gray (#F5F5F5) for a clean and unobtrusive backdrop.
- Accent color: Soft green (#A5D6A7) to highlight key interactive elements and status indicators.
- Body and headline font: 'Inter' for a clean, modern, and readable aesthetic.
- Minimalist layout with clear, logical sections and intuitive placement of elements.
- Simple line icons to represent different functions and states.
